Executes one iteration of the battery check FSM. 

Reads ADC value and determines if battery is low.

Uses ADC readings to determine battery voltage level. If voltage falls below a threshold (\texorpdfstring{$\sim$}{\string~}10.05V), an indicator LED is turned on. When the voltage rises above a recovery threshold (\texorpdfstring{$\sim$}{\string~}10.20V), the LED is turned off. This ensures that the LED won\textquotesingle{}t be constantly switching back and forth when the battery voltage decreases.
