\doxysubsection{Detailed Description}
Bluetooth command handler for robot control. 

This file declares the \doxylink{class_b_l_u_e_t_o_o_t_h}{BLUETOOTH} class, which implements a finite state machine that processes commands received via a Bluetooth module (HC-\/05). It translates commands into motor setpoints for directional control.
