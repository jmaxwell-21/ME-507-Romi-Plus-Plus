\doxysubsection{Detailed Description}
Motor control task with feedback and coordination logic. 

Defines the \doxylink{class_m_o_t_o_r___t_a_s_k}{MOTOR\+\_\+\+TASK} class, which uses encoder feedback and setpoints from various control modules (Bluetooth, Heading, Balance) to regulate motor speed and direction using PID control.
